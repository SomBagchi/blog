# sombagchi.com

Source for [sombagchi.com](https://sombagchi.com) — essays on AI performance
engineering, training at scale, and AI economics. Built with [Quarto](https://quarto.org);
theme adapted (MIT) from the stylesheet of
[How to Scale Your Model](https://jax-ml.github.io/scaling-book/).

- Develop: `quarto preview`
- Publish: push to `main` (GitHub Action renders and deploys to GitHub Pages)
- New essay: copy `posts/2026-07-21-hello-world/` as a template
